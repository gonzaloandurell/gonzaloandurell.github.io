# Portfolio Profesional - Gonzalo Andurell

¡Bienvenido al repositorio de mi página web personal y portfolio profesional! Este sitio web está alojado en [GitHub Pages](https://pages.github.com/) y sirve como presentación de mis proyectos, habilidades y experiencia, principalmente enfocado en el desarrollo de videojuegos (Game Development).

## 🚀 Sobre el Proyecto

Este repositorio contiene el código fuente de mi página web personal (`gonzaloandurell.github.io`). El objetivo principal de la web es:
- Mostrar mis proyectos y aportes en el desarrollo de videojuegos.
- Detallar mis habilidades técnicas y herramientas que domino (como Unity, C#, etc.).
- Facilitar una forma de contacto profesional.

## 🛠️ Tecnologías que se utilizarán

- HTML5
- CSS3 (Diseño moderno y responsive)
- JavaScript (Interacciones y flujos dinámicos)
