Sci-Fi Metal Environment

Maya Substance Painter Unity (Built-In RP) Amplify Shader Editor VFX

Project Overview

This project features a large-scale 3D Sci-Fi environment created from scratch during my coursework. It includes complex geometry such as an intricate vehicle, industrial walls, tables, and heavy machinery. Beyond 3D modeling and texturing, this scene serves as a showcase for custom shaders and particle systems developed in Unity.

Technical Challenges & Solutions:

  • Environment Modeling: Modeled the entire scene in Maya, ensuring clean topology for both the large architectural pieces and the detailed mechanical props (vehicle, machinery).
  • PBR Texturing: Textured assets in Substance Painter, focusing on worn metals, industrial paint, and emissive materials to bring the sci-fi set to life.
  • Custom Shaders (Amplify): Developed interactive node-based shaders using Amplify Shader Editor for the Unity Built-in Render Pipeline, optimizing performance while retaining high visual fidelity.
  • Particle Systems (VFX): Integrated dynamic particle effects that react with the environment and custom shaders, adding depth and motion to the static scene.
